프로그램 기능 정의
1. 데이터를 저장하는 클래스 정의
   Value Object
2. 데이터베이스의 dept 테이블의 데이터를 CRUD기능
3. DAO : Data Access Object
   싱글톤 : 하나의 인스턴스만 생성해서 사용하도록 하는 패턴
   
	1-리스트 출력 -> select 결과를 모아서 반환
				Dept 객체들을 배열 또는 컬렉션으로 수집하고 반환할 때는 이 배열을 보여주면 된다.
	2- 입력 -> insert하기 위해서는 데이터가 필요 . Dept 객체를 받는다. (dept객체는 내가 입력해야 하는 세 변수를 일컫는다.)
	3- 수정 -> update. 데이터 필요. Dept 객체
	4- 삭제 -> delete. deptno가 필요. 숫자를 받으면 된다.( 정수)  
   
4. UI : 1. 입력 2. 검색 3. 수정 4. 삭제 5. EXIT

싱글톤 패턴 : 여러개의 인스턴스를 생성하지 못하도록 하는 디자인 패턴
싱클톤 순서
1. 외부 클래스 또는 인스턴스에서 해당 클래스로 인스턴스를 생성하지 못하도록 처리
2. 클래스 내부에서 인스턴스를 만들고 
3. 메소드를 통해서 반환하도록 처리
